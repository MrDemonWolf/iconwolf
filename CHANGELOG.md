# Changelog

All notable changes to this project will be documented in this file.

## [0.0.6] - 2025-02-11

### Added

- ESLint flat config (`eslint.config.js`) for ESLint v9 with TypeScript support
- CI lint job: runs ESLint and Prettier format check on all PRs and pushes to main
- `format:check` script for CI-friendly Prettier validation
- CLI end-to-end tests (`tests/cli.test.ts`) covering `--version`, `--help`, all flags, and error handling
- Generator integration tests for `--favicon`, `--splash`, multiple flags, custom `--bg-color` with `.icon` input, temp file cleanup, and invalid input error handling
- Unit tests for `applyRoundedCorners` (alpha channel and transparent corner verification)
- Favicon rounded corner pixel-level verification test

### Changed

- CI workflow now runs two parallel jobs: `lint` (ubuntu) and `test` (macOS 14)
- Test count increased from ~40 to 57 across 9 test files

## [0.0.5] - 2025-02-10

### Added

- Favicon rounded corners using Apple-style ~22.37% corner radius (SVG mask)
- `applyRoundedCorners()` utility in `image.ts`
- `resolveDefaultOutputDir()` for automatic `src/` directory detection
- Favicon is now opt-in only via `--favicon` flag (not generated by default)

### Changed

- Default output is now 5 files (icon, android x3, splash) instead of 6
- Default output directory auto-detects `src/` projects (`./src/assets/images/` vs `./assets/images/`)

## [0.0.4] - 2025-02-10

### Fixed

- Build script `sharp` version resolution error (captured version before `cd` into staging directory)

## [0.0.3] - 2025-02-09

### Added

- Apple Icon Composer `.icon` folder support as primary input format
- Icon composer folder tests
- CI test workflow
- Homebrew formula caveats with usage documentation

## [0.0.2] - 2025-02-08

### Fixed

- CLI wrapper to resolve Homebrew symlinks
- Discord invite link updated to mrdwolf.net/discord

### Changed

- README format updated

## [0.0.1] - 2025-02-07

### Added

- Initial release
- Cross-platform app icon generator for Expo/React Native projects
- Generates icon.png, android adaptive icons, splash icon, and favicon
- Sharp-based image processing
- Commander CLI framework
- Homebrew formula distribution
